# 闯关叭叭 · 上架流程

面向本仓库（闯关叭叭 / GoDr）的实操清单。类目与政策以微信公众平台最新规则为准：  
[小程序开放的服务类目](https://developers.weixin.qq.com/miniprogram/product/material.html)

---

## 总览

```
注册小程序 → 配置 AppID → 完善资料/类目/隐私 → 本地自测
→ 开发者工具上传代码 → 提交审核 → 审核通过 → 发布
→（如需）小程序备案
```

当前账号若是 **个人主体**，推荐类目：**教育服务 → 教育信息展示**（无办学许可时的常见选择，**不保证一定过审**）。

## 1. 注册与拿到正式 AppID

1. 打开 [微信公众平台](https://mp.weixin.qq.com/) → 注册 → 选「小程序」
2. 用未绑定过的邮箱注册，管理员微信扫码
3. 主体一般选 **个人**（本项目无服务端、无支付时够用）
4. 登录后台：**开发 → 开发管理 → 开发设置** → 复制 **AppID**  
   本仓库当前 AppID：`wx0b996dbb62bf50cc`（已写入 `project.config.json`）

### 安全提醒

| 可以上 GitHub | 绝不要上 GitHub |
|---------------|-----------------|
| 源码、AppID（前端本来就会暴露） | **AppSecret**、支付密钥、服务器私钥 |

- 本仓库已忽略 `project.private.config.json`，建议把正式 AppID 放在本地该文件或开发者工具项目设置里
- Public 仓库别人能下载代码，**不能**用你的号上传/发布小程序（需要你的微信开发者权限）

## 2. 开发者工具绑定正式 AppID

1. 打开微信开发者工具，导入本仓库根目录（含 `project.config.json`）
2. 详情 → 基本信息 → 将 AppID 改为正式 ID（不要再用 `touristappid`）
3. 确认：
   - `miniprogramRoot` 为 `miniprogram/`
   - 已开启 TypeScript 编译插件
4. 编译 → 真机预览，确认首页 / 练习 / 进度 / 我的可用

---

## 3. 公众平台完善上架资料

路径大致：**设置 → 基本设置 / 服务类目 / 用户隐私保护**（菜单名可能微调）。

### 3.1 基本信息

| 项目 | 建议 |
|------|------|
| 名称 | **闯关叭叭** |
| 简介 | 见下方「可粘贴文案」 |
| 图标 | `assets/miniprogram-avatar.png`（建议 ≥ 144×144） |
| 服务类目 | **教育服务 → 教育信息展示**（个人） |

不要选：培训机构、在线视频课程、游戏/小游戏（门槛高或需额外资质）。

### 3.2 用户隐私保护

本项目：本地存储进度、无登录、不收集手机号。  
隐私说明中写清：**仅本地保存练习进度与错题记录，不上传服务器**。

按后台指引勾选实际用到的接口（如本地存储相关项）。

### 3.3 小程序备案（按平台要求）

国内小程序普遍需要完成备案后才能正式对外服务。按公众平台提示走「小程序备案」，个人主体按身份证等信息填写即可。  
备案服务内容尽量与类目一致（教育信息展示 / 学习工具向），勿填成培训机构。

## 5. 上传代码

在微信开发者工具：

1. 点击右上角 **上传**
2. 填写版本号（如 `1.0.0`）与项目备注（如：首发 · 教育信息展示定位）
3. 上传成功后，代码出现在公众平台 **管理 → 版本管理 → 开发版本**

此时用户还搜不到，只是把包交到微信后台。

---

## 6. 提交审核

1. 公众平台 → **版本管理** → 开发版本 → **提交审核**
2. 勾选/确认类目：**教育服务 - 教育信息展示**
3. 填写审核备注（见下方可粘贴文案）
4. 测试说明写：**无需登录，打开即可使用全部功能**
5. 按要求补充截图（首页、练习页、进度页各一张较稳妥）

审核一般数个工作日；被拒会给原因，按原因改文案/类目/功能后再提。

---

## 7. 发布

审核通过后：

1. 版本管理 → 审核版本 → **发布**
2. 发布后可生成小程序码分享；个人主体搜索曝光可能较弱，主要靠二维码/链接传播

## 9. 常见问题

### Q：选「教育信息展示」一定能过吗？

不能保证。个人主体开放的教育类目很少，这是较贴合的一项；若审核认为互动练习超出「信息展示」，可能被拒。被拒后可：弱化游戏化表述、补充说明「自测工具」、或考虑个体工商户/企业主体后再选类目。

### Q：别人 clone 了我的 GitHub，会更新我的线上小程序吗？

不会。上传/发布必须用你的微信开发者身份。对方最多用代码自己注册新小程序。

### Q：touristappid 能上架吗？

不能。必须换成正式 AppID。

### Q：还要软著、版号吗？

按当前「普通小程序 + 教育信息展示 + 免费无内购」路径，**一般不按游戏类目走**，不主动申请游戏版号。若改成微信小游戏或明确游戏类目，再另议资质。
